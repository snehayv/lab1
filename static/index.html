Projects

  1. Product Recommendations

    • Built a real time product recommendation engine based on browsing patterns and user history for 15+ online retail merchants.
    • Implemented collaborative filtering algorithms over map-reduce to scale for over 100 million users and products with a serving latency of < 400ms.
    • This product is currently live with user coverage of ~90% and is already increasing click-through-rate.
    • Technologies: Java, Hadoop, Cascading, Cassandra Db, AWS EMR and EC2.

  2. Calatrava - Cross Platform Mobile Application Development Framework

    • Part of the core team that built ‘Calatrava’ (calatrava.github.com) – an open-source framework for cross platform mobile application development.
    • Calatrava provides fidelity of native user experience (by leveraging platform specific components) and minimizes cost (by sharing logic and views across platforms).
    • Used Calatrava to develop Apps on Android, iOS, BB & Mobile Web for a leading US airline.
    • Technologies: HTML5, Coffee Script (JS), Single Page Architecture, Node JS, Objective-C, Java.

  3. Dating Website matching algorithm

    • Completely revamped the core algorithm, user experience & performance of a dating website for a leading news and media company in UK.
    • Developed a weighted scoring algorithm to match prospective partners based on statistics, user data & behavior. This significantly improved the revenue and the users’ interactions observed by analytics.
    • Also worked on front-end development, performance improvements, interactive messaging & several other key features of the website.
    • Technologies: Scala, Django-python (client side templating), Solr (text search engine), Mongo Db.
